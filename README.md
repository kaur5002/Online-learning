# Online Learning Platform

## Overview
The Online Learning Platform is a web-based application designed to support online education through structured content delivery and user interaction. The project focuses on modern web development practices, system design, and usability.

This project was developed as part of an academic course in Information Systems and demonstrates full-stack development concepts.

---

## Features
- User-friendly interface for browsing and accessing learning content
- Client-side data fetching with caching and synchronization
- Responsive and accessible UI design
- Form handling with validation
- Modular and reusable component architecture

---

## Technologies Used
- **Next.js (React)** – routing, server-side logic, and API routes  
- **TanStack Query** – client-side data fetching and caching  
- **Tailwind CSS** – responsive styling  
- **shadcn/ui (Radix primitives)** – reusable UI components  
- **react-hook-form & Zod** – form handling and validation  
- **Axios** – API communication  
- **GitHub** – version control

---

## Project Structure
- `app/` – application routes and pages  
- `components/` – reusable UI components  
- `hooks/` – custom React hooks  
- `lib/` – shared utilities  
- `prisma/` – database-related configuration  
- `public/` – static assets  

---

## Purpose
This project was developed to apply concepts such as:
- Web-based information systems
- Client–server interaction
- UI/UX principles
- Software structure and maintainability
